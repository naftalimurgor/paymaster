## Cirlce Paymaster example

An example showing gasless USDC transfers using circle paymaster

## Getting started

1. Clone this repository

```sh
git clone https://github.com/naftalimurgor/paymaster
```

2. Install dependencies:

```sh
npm i
```

3. set env secrets in `.env`. A sample is provided in `.env.example`
4. Grab some tesnet USDC here: https://faucet.circle.com/
5. Run example:

```sh
node src/index.js
```

More details: https://developers.circle.com/stablecoins/quickstart-circle-paymaster#12-initialize-clients-and-smart-account
